Tutorial NodeJS, Express, Atom, Pandoc, GitHub Desktop

NodeJS

NodeJS es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación ECMAScript, con I/O de datos en una arquitectura orientada a eventos basado en V8 de Google.

Para instalar NodeJS vamos a su página de instalación y siguiendo las instrucciones del asistente de instalación completamos el proceso para tener NodeJS en nuestro ordenador:

Express

Express es una infraestructura de aplicaciones web Node.js mínima y flexible que proporciona un conjunto sólido de características para las aplicaciones web y móviles. Para instalarlo en nodeJS lo haremos con el siguiente comando:

$ npm install express --save

Atom

Atom es un editor de texto que puedes modificar y personalizar a tu gusto, orientado a escribir código en una gran variedad de lenguajes. Una vez descargado el paquete de instalación es cuestión de sólo seguir el asistente para instalar satisfactoriamente el programa.

Pandoc

Pandoc es un conversor universal de documentos y nos permite cambiar entre una gran variedad de formatos de documentos de texto. Nosotros lo usaremos para convertir nuestro código Markdown en html.

Esta conversión se realiza en la consola de windows (en nuestro caso) y una vez en la carpeta donde se encuentra nuestro fichero markdown, la sintaxis es la siguiente:

$ pandoc -f formatodeentrada -t formatodesalida nombrefichero.md -o nombreficherosalida.html

En nuestro caso usaremos la siguinte línea de consola:

$ pandoc -f markdown -t html tutorial.md -o tutorial.html

Si no pusiéramos el fichero de salida ya sea en .html o en otro tipo de archivo, el pandoc nos mostraría la conversión por consola.

Github Desktop

Github Desktop es una aplicación del servicio de repositorios online Git para poder tener de manera visual una estructura completa o parcial de nuestros repositorios de la nube pudiendo clonarlos y realizar trabajo en local. Posee distintas cualidades que hacen que el trabajo se pueda realizar más rápido o al menos de una forma más sencilla y atractiva que por consola, como puede ser la realización de commits. Para empezar a utilizar GithubDesktop solo tenemos que descargarnos el instalador y seguir los pasos del asistente.
